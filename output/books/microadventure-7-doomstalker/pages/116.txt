What the Program Does
BRUTE has used a key substitution code
to encode its message. Orion has one version
of the BRUTE decoder program that uses the
key "TEAMS." When the message is de-
coded by this program, something strange
happens. Some of the letters seem to be de-
coded correctly, while others are still gar-
bage. It looks as if Orion has the right program,
but the wrong key. Can you transpose the let-
ters in the key to make another key word and
try decoding with that?
