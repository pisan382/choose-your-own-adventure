Functions are ready-made routines that per-

sort of like having a key on a calculator that

INT is a function that changes any number that
you supply into a whole number or integer. For
example INT(4.5) will return the value 4. For

any fractions and supplies you with the whole
number.
LEN is a function that tells you the number of
characters in a string of letters, numbers, or
other symbols. For example, if a variable string

Random Number Generator This function,
