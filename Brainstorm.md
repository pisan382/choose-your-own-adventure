# Choose Your Own Adventure

## CSS 382 - Group 6: Alec Situ, Hyobin Yook 

### Goal
web app that users (authors) can upload story segments and link them into a branching graph (story graph) and readers readers play through them by making choices.


### part 1: author features
*Interactive graph nodes*
- **Create, edit, and delete** story segment nodes 
- **Connect** nodes by adding labeled choices (e.g. "Go to the cave", instead of page 3)
- **Visualize** the full story as an interactive graph
- Set a start node and mark terminal (ending) nodes
- Save and load stories as JSON

### part 2: reader features
- **Read/display** one story segment at a time
- Click a choice to advance to the next segment
- Reset to the beginning
- (maybe?) See a progress indicator or visited path
